# Блянер — Финальная спецификация (Обновлено 12 апреля)

## 1. Авторизация
- Email + пароль (Supabase Auth)
- Google OAuth
- Telegram Login Widget

## 2. Навигация и Шапка
- **Саб-хэдер:** Логотип "Бля, {имя}" в левом сайдбаре.
- **Дропдаун профиля:** Клик по аватарке в шапке → Профиль, Настройки, Экспорт, Тема, Выйти.
- **Трекер времени (Шапка):** Секундомер + прогресс-кольцо (выполнено задач из общего количества на сегодня). Если сегодня добавлена/выполнена задача — счетчик в шапке обновляется (например, 3/7 → 4/7).

## 3. Экран «Неделя»
- **Логика задач:** Клик по задаче → зачеркивается (выполнено). Двойной клик → редактирование названия. Long press → выбор проекта.
- **Ввод времени на Неделе:** (По комментарию со скрина: "как записать время выполнения задачи со стр Неделя?") → Добавим возможность кликнуть на карточку или через hover-меню вписать затраченное время.
- **Виджеты сайдбара:** Сон (бары + кнопки), На неделю, Проекты.

## 4. Экран «День»
- **Карточка задачи:** 
  - Редактирование названия — двойной клик (как на Неделе).
  - Смена проекта — кликабельна вся зона проекта (кружок и текст). При ховере подсвечивается (hover:opacity-70).
- **Раскрытая карточка (Аккордеон):**
  - Подзадачи (чеклист + инпут).
  - Заметка (одна строка).
  - Цветной кружок проекта и название (клик → палитра для смены).
  - Бейдж времени (клик → инлайн редактирование).
  - Кнопка-меню (⋯): Удалить, Перенести на другой день, Дублировать.
- **Секция «Выполнено»:** При наведении на галочку/кружок — фиолетовая подсветка.
- **Сайдбар:** Сон (прогресс-бар + время + КНОПКИ Сплю/Встала), Итого, На неделю, Проекты.
  - Тултип в блоке Сон: Иконка "?" в правом верхнем углу, при наведении всплывает подсказка: "Нажми Сплю перед сном и Встала утром. Блянер посчитает сколько ты спала и покажет статистику по дням."
  - Бары сна кликабельны/ховерабельны → при наведении тултип с длительностью и временем (Пн: 6.2ч, 04:30 - 10:42).

## 5. Экран «Месяц»
- **Сайдбар:** Нужно продумать логику виджетов. Предлагаю: Статистика месяца (кол-во выполненных задач), Сон (мини-график), Дедлайны (список).

## 6. Профиль
- Статистика (всего создано/выполнено/затрачено часов).
- Стрик.
- Фокус за всё время (прогресс-бары по проектам).
- Продуктивность.

## 7. Настройки
- Левая колонка: Виджеты (вкл/выкл).
- Правая колонка: Аккаунт, Смена пароля, Экспорт, Удаление аккаунта.
