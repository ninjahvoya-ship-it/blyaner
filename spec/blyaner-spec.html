Блянер — Спецификация фичей

«Бля, работает.»

1. Авторизация

1.1 Вход

  • Email + пароль — стандартный вход
  • Google OAuth — один клик через Supabase Auth
  • Telegram Login Widget — для русской ЦА
  • Табы Вход/Регистрация на одной странице

1.2 Регистрация

  • Поля: Email, Пароль, Повторите пароль
  • Имя НЕ спрашиваем (подтягивается из Google/Telegram автоматически)
  • После регистрации → сразу на пустую Неделю

1.3 Забыли пароль

  • Ввод email → отправка ссылки → success-экран «Письмо отправлено»
  • Ссылка «Отправить ещё раз»

1.4 Персонализация заголовка

  • Если имя есть → сайдбар: «Бля, {имя}», заголовки: «Твоя неделя», «Твой день» и т.д.
  • Если имени нет → сайдбар: «Блянер», заголовки: «Моя неделя», «Мой день» и т.д.

2. Навигация

2.1 Сайдбар (левая панель, всегда видна)

  • Логотип + название (Бля, {имя} / Блянер)
  • Мини-календарь (листание месяцев ←/→, клик по дню → вид «День»)
  • Виджеты (зависят от экрана и настроек вкл/выкл):
  • Сон
  • Итого (День)
  • На неделю / На месяц
  • Проекты (Неделя/День)
  • Дедлайны (Месяц)
  • Незавершённые (Месяц)

2.2 Шапка (верхняя панель)

  • Строка 1: Заголовок страницы + переключатель День/Неделя/Месяц + секундомер + виджет задач (прогресс-кольцо) + аватар
  • Строка 2: Навигация по датам (← дата →)
  • Разделительная линия между строками

2.3 Аватар (лаймовый кружок с инициалом)

  • Клик → дропдаун: Профиль, Настройки, Экспорт, Тема, Выйти

3. Экран «Неделя» (главный)

3.1 Сетка

  • 7 колонок (Пн-Вс), заголовки с датами
  • Сегодняшний день подсвечен (лаймовый фон заголовка + метка «сегодня»)
  • Колонки белые, полная высота фрейма

3.2 Задачи в колонках

  • Цветные карточки (цвет = проект)
  • Подпись: название проекта + время
  • Один клик → задача выполнена/не выполнена (toggle, зачёркивание + opacity)
  • Двойной клик → редактирование текста (inline, Enter сохраняет)
  • Long press → меню выбора проекта (палитра кружочков)
  • Drag & drop → перетаскивание между днями
  • Крестик × → удаление (при ховере) + тост «Отменить» (5 сек)

3.3 Создание задачи

  • Инпут + задача... внизу каждой колонки
  • Enter → создаётся белая карточка с фиолетовой обводкой, подпись «Без проекта»
  • Потом long press → назначить проект

3.4 Виджеты сайдбара (Неделя)

  • Мини-календарь
  • Сон (бары по дням, кнопки Сплю/Встала)
  • На неделю (чеклист целей + кнопка +)
  • Проекты (список + кнопка +)

4. Экран «День»

4.1 Список задач

  • Карточки в ряд (не таймлайн по часам!)
  • Каждая карточка: цветная полоска слева (проект) + ▶️ Play + название + проект + бейдж времени + ○ чекбокс + ⋯ меню
  • Активная задача (с таймером): розовый бордер + пульсирующий бейдж ⏱ 01:23
  • Клик по карточке → разворачивается: подзадачи (чеклист) + заметка + инпут подзадачи

4.2 Чекбокс ○ (справа)

  • Клик → задача выполнена, уезжает в секцию «Выполнено»

4.3 Секция «Выполнено»

  • Свёрнутая секция внизу (▼ Выполнено · 3)
  • Задачи зачёркнуты, полупрозрачны, с лаймовой галочкой

4.4 Создание задачи

  • Инпут + Добавить задачу... между активными и выполненными

4.5 Виджеты сайдбара (День)

  • Мини-календарь
  • Сон сегодня (прогресс-бар + время засыпания/пробуждения)
  • Итого (выполнено X/Y, затрачено Xч Yм, фокус-полоска)
  • На неделю
  • Проекты

5. Экран «Месяц»

5.1 Сетка 5×7

  • Заголовки дней (Пн-Вс)
  • Каждая ячейка: число + цветные точки задач + название (первые 3, потом «+N ещё»)
  • Сегодня: лаймовая заливка + лаймовый кружок вокруг числа
  • Прошлый/следующий месяц: серый фон, полупрозрачные числа
  • Дедлайн: красная точка рядом с задачей
  • Клик по дню → переход на экран «День»

5.2 Виджеты сайдбара (Месяц)

  • Статистика месяца (задачи, время, фокус-полоска)
  • Сон (мини-график за 14 дней, бары зелёные/розовые, среднее)
  • На месяц (чеклист целей, вместо «На неделю»)
  • Дедлайны (список ближайших с датами)
  • Незавершённые (задачи с прошлых дней)
  • Проекты УБРАНЫ с Месяца

6. Секундомер

6.1 Привязанный трекинг

  • Кнопка ▶️ на карточке задачи → секундомер стартует и привязан к этой задаче
  • В шапке: время + название активной задачи
  • Кнопка ⏸ Pause → остановка
  • Время записывается в задачу

6.2 Свободный трекинг

  • Кнопка ▶️ в шапке (без привязки) → свободный отсчёт
  • При остановке → вручную вписать время куда нужно

6.3 Ручной ввод времени

  • Клик по бейджу времени на карточке → инлайн-редактирование
  • Формат: «45м», «1ч 20м»

7. Проекты

7.1 Создание

  • Кнопка + в сайдбаре → инлайн-форма (НЕ модалка):
  • Инпут названия
  • Палитра из 8 цветов (purple, lime, pink, yellow, blue, orange, teal, red)
  • Кнопка «Создать» / Enter
  • × для отмены / Esc
  • Максимум 10 проектов

7.2 Редактирование

  • Карандаш при ховере в списке проектов (Профиль) → смена названия/цвета

7.3 Назначение задаче

  • Long press на карточке задачи → палитра кружочков

8. Сон

8.1 Трекинг

  • Кнопка «Сплю» → запоминает время засыпания
  • Кнопка «Встала» → запоминает время пробуждения
  • Автоматический расчёт длительности

8.2 Отображение

  • Неделя: бары по дням (зелёный >5ч, розовый <4ч) + среднее + кнопки
  • День: прогресс-бар + точное время + кнопки
  • Месяц: мини-график 14 дней + среднее
  • Тултип: при ховере на бар → «Пн: 6.2ч (04:30–10:42)»
  • Подсказка: иконка ? — объяснение фичи для новых юзеров

8.3 Настройки

  • Можно выключить в Настройки → Виджеты → Трекер сна

9. Дедлайны

9.1 Установка

  • Меню ⋯ на карточке задачи → «Дедлайн» → выбор даты

9.2 Отображение

  • День/Неделя: красный бейдж 🚩 с датой на карточке
  • Месяц: красная точка рядом с задачей в ячейке дня
  • Сайдбар (Месяц): список ближайших дедлайнов с цветом и датой

10. Меню задачи (⋯)

Дропдаун при клике на три точки:

  • Перенести (на другой день)
  • Дублировать
  • Дедлайн → выбор даты
  • —— разделитель ——
  • Удалить (красный)

11. Подзадачи

  • Видны при раскрытии карточки (экран День)
  • Чеклист с круглыми чекбоксами (лаймовая заливка при выполнении)
  • Инпут + подзадача... для добавления
  • Заметка (одна строка, опциональная, иконка карандаша)

12. Профиль

  • Аватар (лаймовый квадрат с инициалом)
  • Имя + email + дата регистрации
  • Статистика: 4 плитки (создано, выполнено, затрачено, стрик)
  • Стрик: число + ряд иконок-огоньков (ph-flame, закрашенные/серые) + рекорд
  • Фокус за всё время (полоска + разбивка по проектам с часами)
  • Проекты (список с количеством задач + карандаш + + Добавить)
  • Продуктивность: лучший день, пик активности, среднее задач/день

13. Настройки

Двухколоночная раскладка:

Левая колонка: Виджеты (toggle вкл/выкл)

  • Трекер сна
  • Стрик
  • Секундомер
  • Дедлайны
  • Незавершённые

Правая колонка:

Аккаунт:

  • Изменить имя
  • Изменить email
  • Сменить пароль (→ модалка)
  • Часовой пояс

Данные:

  • Экспорт (CSV / JSON) → модалка выбора формата

Опасная зона:

  • Удалить аккаунт (→ модалка подтверждения с вводом «УДАЛИТЬ»)

Внизу: Блянер v1.0 · Бля, работает.

14. Пустые состояния

Пустая неделя (первый запуск)

  • Пустые колонки с инпутами
  • Сегодняшний день: «Бля, пусто! Впиши первую задачу» (по центру-вверх)
  • Сайдбар: пустой сон, «Пока пусто. Нажми + чтобы добавить цель», «Создай первый проект»

Пустой день

  • «Бля, свободный день! Или ты забыла добавить задачи?»
  • Иконка кружки кофе
  • Большой инпут по центру

15. Тост «Отменить»

  • Появляется после удаления задачи
  • Тёмная плашка внизу экрана: «Задача удалена — [Отменить]»
  • Лаймовая кнопка
  • Автоматически исчезает через 5 секунд
  • Клик «Отменить» → задача возвращается на место

16. Технические решения

Стек

  • Frontend: Next.js (React) + Tailwind CSS
  • Backend/Auth/DB: Supabase (Auth + PostgreSQL + Realtime)
  • Хостинг: Vercel (бесплатный тир)
  • Иконки: Phosphor Icons Light (ph ph-*), заливные для акцентов (ph-fill)
  • Шрифт: Inter (400, 500, 600, 700, 800)

Палитра

  • Sidebar: #8B7EC8
  • Lime: #D4E84D / #B8CC35
  • Main BG: #F4F3F8
  • Surface: #FFFFFF
  • Text Dark: #2D2B3D
  • Text Muted: #8E8BA0
  • Grid Line: #ECEAF4
  • Purple Card: #B8ADE8
  • Проекты: purple, lime, pink, yellow, blue, orange, teal, red

Принципы

  • Без эмодзи-иконок (только Phosphor Icons)
  • Инлайн-ввод (type + Enter) вместо модалок где возможно
  • Один клик = самое частое действие
  • Тёплый минимализм

Переходы между экранами

Основная навигация (табы в шапке)

День ←→ Неделя ←→ Месяц
  • Клик по табу → мгновенный переход
  • Активный таб подсвечен фиолетовым
  • URL: /day, /week, /month

Из Месяца в День

Месяц → [клик по ячейке дня] → День (эта дата)

Из мини-календаря (сайдбар)

Мини-календарь → [клик по числу] → День (эта дата)

Навигация по датам (стрелки ←/→)

День:   ← вчера | завтра →
Неделя: ← прошлая неделя | следующая →
Месяц:  ← прошлый месяц | следующий →

Аватар → Профиль (/profile)
       → Настройки (/settings)
       → Экспорт (модалка)
       → Выйти (→ Вход)

Флоу пользователя

Первый запуск (новый юзер)

1. Вход/Регистрация
   ├─ Email + пароль → Регистрация
   ├─ Google → OAuth → автологин
   └─ Telegram → Widget → автологин
        │
2. Попадает на ПУСТУЮ НЕДЕЛЮ
   │  Сайдбар: пустой сон, пустые цели, «Создай первый проект»
   │  Сегодняшний день: «Бля, пусто! Впиши первую задачу»
   │
3. Вводит первую задачу (Enter)
   │  → Белая карточка «Без проекта»
   │
4. Создаёт первый проект (+ в сайдбаре)
   │  → Инлайн-форма: название + цвет → «Создать»
   │
5. Назначает проект задаче (long press)
   │  → Карточка окрашивается
   │
6. Готово! Пользователь понял механику

Ежедневный флоу

1. Открывает Блянер
   │
2. Видит «Бля, Ань — Твоя неделя»
   │
3. Добавляет задачи на сегодня
   │  Инпут → Enter → карточка
   │
4. Работает над задачей
   │  ▶️ на карточке → секундомер тикает
   │  ... работает ...
   │  ⏸ → время записалось
   │
5. Завершает задачу
   │  Неделя: клик → зачёркнулась
   │  День: ○ чекбокс → уехала в «Выполнено»
   │
6. Перед сном
   │  «Сплю» → записано время засыпания
   │
7. Утром
   │  «Встала» → записано, виджет сна обновился

Флоу задачи

Создание → [инпут + Enter]
    │
    ├─ Назначить проект → [long press → палитра]
    ├─ Добавить время → [клик по бейджу → инлайн]
    ├─ Запустить таймер → [▶️]
    ├─ Добавить подзадачи → [раскрыть карточку (День)]
    ├─ Поставить дедлайн → [⋯ → Дедлайн → дата]
    ├─ Перенести на другой день → [drag & drop / ⋯ → Перенести]
    ├─ Дублировать → [⋯ → Дублировать]
    ├─ Выполнить → [клик / ○]
    └─ Удалить → [× / ⋯ → Удалить] → тост «Отменить»
