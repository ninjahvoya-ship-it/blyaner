# Блянер

> Бля, работает.

Планировщик задач с секундомером, трекером сна и статистикой.

## Стек
- Next.js 14 (React + TypeScript)
- Tailwind CSS
- Supabase (Auth + Database + Realtime)
- Phosphor Icons
- Vercel (хостинг)

## Запуск
```bash
npm install
cp .env.local.example .env.local
# заполни Supabase credentials
npm run dev
```
